# Capstone Project Requirements

## Overview
Create an embedded system simulator that integrates concepts from all months of the C Embedded Programming course.

## Features
- Command interpreter for user input
- Sensor data processing and logging
- State machine for system control
- Modular design with multiple files

## Requirements
- Use structs, pointers, and dynamic memory
- Implement state machines
- Handle file I/O
- Demonstrate embedded programming concepts

## Deliverables
- Complete C program
- Documentation
- Test cases
